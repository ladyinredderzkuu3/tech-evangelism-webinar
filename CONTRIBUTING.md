# Mitwirkungsrichtlinien

Vielen Dank, dass Sie zu unserem Projekt beitragen möchten! Hier sind einige Richtlinien für die Mitwirkung:

## Wie kann man beitragen?
- Öffnen Sie einen Issue, um einen Fehler zu melden oder eine Funktion vorzuschlagen.
- Forken Sie das Repository, erstellen Sie einen Branch für Ihre Änderungen und reichen Sie einen Pull-Request ein.

## Code-Standards
- Halten Sie sich an die vorhandene Code-Struktur und -Formatierung.
- Fügen Sie Kommentare zu Ihrem Code hinzu, um dessen Funktion zu erklären.

## Weitere Informationen
Lesen Sie bitte die [README.md](README.md) für weitere Informationen über das Projekt.
